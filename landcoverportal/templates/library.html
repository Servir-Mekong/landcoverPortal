Land Cover Mapping Google Earth Engine Scripts

A large number of indices were calculated from surface reflectance composites. These were combined with training data in the machine learning algorithm. The machine learning algorithm produced primitives which are probability maps for each land use category. An assemblage script was used to create land cover maps. The accuracy assessment was done with validation data.
